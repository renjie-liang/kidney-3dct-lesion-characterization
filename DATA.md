# Data

This repository does not contain private clinical data.

## UF Health Dataset

The UF Health kidney CT dataset described in the manuscript contains de-identified clinical CT volumes and radiology-report-derived annotations from a single academic medical center. This dataset is not publicly available because of patient privacy restrictions and institutional data-use requirements.

## KiTS23

The KiTS23 dataset is publicly available from the KiTS challenge:

<https://kits-challenge.org/kits23/>

Future release scripts will describe how KiTS23 was processed for external validation without redistributing the dataset itself.

## Privacy Notice

Do not commit protected health information, radiology reports, patient identifiers, private annotations, or institution-restricted data to this repository.
